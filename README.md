# Genshin Impact Game Guide

English-language Genshin Impact game guide website — character builds, F2P teams, spiral abyss strategies, and meta analysis.

**Live at:** [https://12312131.github.io/Genshin-Impact/](https://12312131.github.io/Genshin-Impact/)

## Structure

- `index.html` — Homepage with all 15 guides organized by category
- `guide/` — Individual article pages
- `css/style.css` — Single stylesheet, dark Genshin-accented theme

## Categories

| Category | Articles |

## Tech

Pure static HTML/CSS. No JavaScript, no frameworks, no build step. Deployed via GitHub Pages.
